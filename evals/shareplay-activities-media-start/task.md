# SharePlay Movie Activity Implementation

## Problem/Feature Description

A SwiftUI movie app needs a SharePlay implementation plan that lets people start a shared watch session, join incoming sessions, and keep AVPlayer playback synchronized. The team wants Swift-oriented guidance and code snippets, not a full Xcode project.

## Output Specification

Create `shareplay-movie-activity.md` with setup steps, a GroupActivities activity type, session observation and cleanup, a SwiftUI start path, and AVPlayer coordination guidance. Keep custom messaging scoped to app-specific state that AVPlayer does not synchronize.
