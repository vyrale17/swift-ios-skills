# Habit Tracker Launch Listing

## Problem/Feature Description

A small team is preparing the first App Store launch for a habit tracker. The product is polished enough to submit, but the listing still reads like internal feature notes. The team needs a launch-ready ASO plan that helps people find the app and helps product-page visitors understand why they should install it.

The app tracks daily habits, streaks, flexible reminders, simple progress charts, and routine templates. The brand name is not yet known. Paid search campaigns will target habit tracking, streak motivation, and daily routines.

## Output Specification

Create `aso-plan.md` with a practical launch plan covering the main product page, ad-specific product pages, event ideas, localization approach, testing plan, and review-prompt timing. Keep the plan concise enough for a founder and designer to act on.
