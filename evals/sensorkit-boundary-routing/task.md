# Fitness App Sensor API Recommendation

## Problem/Feature Description

A consumer fitness app wants live step counts, tilt-based workout interactions,
and heart-rate summaries from user workouts. The team noticed SensorKit includes
motion and health-adjacent sensors and is considering using it as the main
sensor layer so they can avoid stitching together multiple Apple frameworks.

## Output Specification

Create `fitness-sensor-api-recommendation.md` with an API recommendation and
implementation direction. Explain which Apple frameworks should own the live
motion and workout-data parts, and whether SensorKit belongs in this product
plan.
