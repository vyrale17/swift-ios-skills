# Shareable Place Preview

## Problem/Feature Description

A travel-planning app wants to generate a compact place preview for each saved destination. The preview should include a noninteractive map image suitable for sharing, a street-level visual preview when the platform has one, and a simple route context from the user's current area to the place.

Design the MapKit implementation. The team is not asking for a full app, but they need a clear plan and representative Swift snippets that avoid turning this into a heavyweight map screen.

## Output Specification

Create `place-preview-plan.md` with:

- The MapKit APIs you would use for the static map image, street-level preview, and route context.
- Representative Swift snippets for each part.
- Notes on task cancellation, errors/no-data fallbacks, and avoiding unnecessary live location work.
