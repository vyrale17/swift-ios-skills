# Account Security Scope Review

## Problem/Feature Description

A team asks the authentication guidance to design the entire account security layer for an iOS app. Their proposed scope includes Sign in with Apple, passkeys, Password AutoFill, OAuth, refresh-token storage, CryptoKit encryption, Secure Enclave key management, certificate pinning, biometric unlock, and OWASP MASVS mapping.

## Output Specification

Write a concise scope review that tells the team what the authentication guidance should cover and what should be handed off to sibling security work. Keep it practical: the team needs enough boundary guidance to route implementation tasks without expanding authentication into a full application security program.
