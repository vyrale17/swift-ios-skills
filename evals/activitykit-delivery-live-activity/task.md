# Delivery Live Activity Implementation Review

## Problem/Feature Description

The delivery team is adding a real-time order tracker for an iOS app. They need a design note that an iOS engineer can use to implement the feature without building a full Xcode project. The experience should cover the app-side data model, lifecycle, visible surfaces, server-driven updates, and cleanup behavior for delivery completion or cancellation.

## Output Specification

Create `delivery-live-activity.md` with Swift-oriented implementation guidance, code snippets where useful, and a short review checklist. Do not create an Xcode project.
