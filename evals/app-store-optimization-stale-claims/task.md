# ASO Guidance Accuracy Review

## Problem/Feature Description

A product marketer copied an older ASO checklist into the launch doc. The team is worried some of it may be stale or too absolute, and they want a correction pass before the doc becomes the standard for future releases.

Draft note to review:

> Custom Product Pages are capped at 35 and only work through direct links. Product Page Optimization compares one treatment in the default locale only. Put repeated keywords at the start of the keyword field because position is a ranking factor. Add more keywords to the description. In-app event metadata is indexed like the keyword field. Japan and Korea also index English keyword fields. Ratings below 4.0 always tank conversion.

## Output Specification

Create `corrections.md` with a line-by-line correction. Keep each item actionable and avoid replacing unsupported claims with new unsupported claims.
