# NaturalLanguage Text Analysis Pipeline

## Problem/Feature Description

An iOS inbox feature needs to analyze message text. It should detect the
message language, tokenize words, extract people and organizations, compute a
sentiment score, and find semantically similar keywords for search suggestions.

## Output Specification

Create a file named `natural-language-text-analysis.md` containing:

- A Swift-oriented implementation outline using Apple's NaturalLanguage
  framework.
- A review checklist covering language detection, tokenization, tagging,
  sentiment, embeddings, thread safety, and unavailable assets/languages.
- Boundary notes for work that should not be handled by NaturalLanguage.

Do not create an Xcode project. Keep the answer as implementation guidance and
snippets only.
