# PermissionKit Significant App Update Review

## Problem/Feature Description

A children's app wants to use
`SignificantAppUpdateTopic(description: "We improved the app")` on devices
running iOS 26.0, skip response observation, and use the same flow for every
terms-of-service wording tweak.

Review the plan and provide the minimal corrected PermissionKit guidance.

## Output Specification

Create a file named `permissionkit-significant-update.md` containing:

- Availability requirements for the significant-update topic and ask flow.
- Guidance for deciding whether an update is significant.
- Better description-writing guidance and an example.
- Response handling and boundary notes.
